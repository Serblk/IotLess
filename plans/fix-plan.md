# План правок

## 1. PDF не открываются

**Диагноз:** PDF.js использует Range-запросы. npx serve может не поддерживать Range headers для кириллических путей. encodeURI() некорректно обрабатывает пробелы в относительных путях.

**Правка в script.js:**
- pdfjsLib.getDocument() — добавить disableRange:true, disableStream:true, disableAutoFetch:true
- Заменить encodeURI(item.file) на new URL(item.file, window.location.origin).href
- Более информативная ошибка в catch
